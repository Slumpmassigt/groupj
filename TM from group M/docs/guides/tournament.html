Tournament Manager¶

How it works¶

The tournament manager is used like this.

# Create a new tournament manager
tm = TournamentManager(create_game, run_game)

# Show the main menu
tm.start()

The TM shows the main menu where the user can choose which tournament type to play and how many players to include. Once selected, the tournament starts and the games will be played.
