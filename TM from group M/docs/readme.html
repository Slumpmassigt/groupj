Nine Men’s Morris¶

This project is a Nine Men’s Morris implementation in Python for the class Software Engineering and Project Management at Uppsala University.

Getting Started¶

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

Prerequisites¶

pipenv is used to manage dependencies and can be installed with

pip install pipenv

Installing¶

Install required python packages

pipenv install

Activate the virtual python environment with

pipenv shell

Tests¶

The automated unit tests are written using pytest.

Run unit tests¶

Run the unit tests and also check the code style and python import ordering.

make test

Run coding style tests¶

Only check the coding style and python module import ordering.

make style

Documentation¶

API documentation and the user guide can be generated with

make docs

and is available in docs/_build/html.

Component-specific documentation¶

Generate documentation with only the Tournament Manager included

make docs-tm

Generate documentation with only the Game Platform included

make docs-gp
